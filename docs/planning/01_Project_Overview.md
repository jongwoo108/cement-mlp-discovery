# 01_Project_Overview.md

> **탄소중립 시멘트 결합재의 AI 기반 계산 설계**

---

## 🎯 프로젝트 요약

이 프로젝트는 기계 학습 포텐셜(MLP)과 분자 동역학(MD) 시뮬레이션을 활용하여 탄소중립 시멘트 결합재를 발굴하는 계산 프레임워크를 개발합니다. 기계적 성능을 유지하면서 시멘트 산업의 CO₂ 배출량을 90% 감소시키는 것을 목표로 합니다.

**상태**: 1단계 완료 (개념 증명)

---

## 🌍 문제 정의

### 기후 문제

- **전 세계 영향**: 시멘트 산업은 전 세계 CO₂ 배출량의 8%를 차지
- **공정 배출**: 90%는 1,450°C 석회석 클링커화 과정에서 발생
- **규모**: 연간 40억 톤 이상의 시멘트 생산
- **비용**: 기존 R&D는 배합당 5-10년 소요

### 왜 중요한가

```
현재 포틀랜드 시멘트 생산:
CaCO₃ → CaO + CO₂ (하소, 1,450°C)
         ↓
    시멘트 톤당 900 kg CO₂
```

**목표**: 30-40 MPa 강도를 유지하면서 톤당 100 kg CO₂ 미만을 생성하는 대안 찾기

## 🔬 연구 접근법

### 다중 스케일 계산 프레임워크

```
┌────────────────────────────────────────────────────────────┐
│ 스케일 1: 원자 수준 (옹스트롬)                              │
│ ├─ 도구: CHGNet 기계 학습 포텐셜                           │
│ ├─ 작업: 구조 최적화, 에너지 예측                          │
│ └─ 출력: 최적화된 기하 구조, 형성 에너지                   │
├────────────────────────────────────────────────────────────┤
│ 스케일 2: 나노 스케일 (나노미터)                            │
│ ├─ 도구: 분자 동역학 (ASE + CHGNet)                        │
│ ├─ 작업: 수화 반응, C-S-H 젤 형성                          │
│ └─ 출력: 반응 메커니즘, 결합 구조                          │
├────────────────────────────────────────────────────────────┤
│ 스케일 3: 마이크로 스케일 (마이크로미터)                    │
│ ├─ 도구: 조립립 MD / 연속체 모델                           │
│ ├─ 작업: 기계적 특성 예측                                  │
│ └─ 출력: 강도, 탄성률, 내구성 지표                         │
├────────────────────────────────────────────────────────────┤
│ 스케일 4: AI 발굴                                          │
│ ├─ 도구: MatterGen (생성 모델)                             │
│ ├─ 작업: 새로운 조성 생성                                  │
│ └─ 출력: 새로운 후보 재료                                  │
└────────────────────────────────────────────────────────────┘
```

### 핵심 혁신: MLP 가속 발굴

**기존 DFT 접근법**:
- 정확도: 높음 (참조 표준)
- 속도: 구조 1개 = 1-24시간
- 처리량: 월 ~10개 재료

**우리의 MLP 접근법**:
- 정확도: DFT 대비 0.15% 오차
- 속도: 구조 1개 = 1-5초
- 처리량: **하루 1000개 이상 재료**

**속도 향상**: 1000-10,000배 빠름

## 📊 연구 질문

### 주요 질문

1. **수화 메커니즘**
   - 원자 수준에서 C₃S는 물과 어떻게 반응하는가?
   - C-S-H 젤 형성 경로는 무엇인가?
   - 수화 동역학을 예측할 수 있는가?

2. **조성 최적화**
   - 어떤 Ca/Si 비율이 강도를 최대화하는가?
   - Al, Fe, Mg 치환이 특성에 어떤 영향을 미치는가?
   - 하소 없이 결합재를 설계할 수 있는가?

3. **대체 결합재**
   - 철강 슬래그 + CO₂ 광물화?
   - 지오폴리머 (알칼리 활성화)?
   - 칼슘 설포알루미네이트(CSA) 시멘트?
   - 비산회 블렌드?

4. **성능 예측**
   - MLP로 압축 강도를 예측할 수 있는가?
   - 어떤 원자 특성이 내구성과 상관관계가 있는가?
   - 수천 개의 후보를 효율적으로 스크리닝하는 방법은?

## 💡 기대 영향

### 과학적 기여

1. MLP를 사용한 C₃S 수화의 **첫 번째 원자 수준 모델**
2. 대체 결합재 특성 **데이터베이스**
3. 시멘트를 위한 **ML 기반 발굴** 프레임워크
4. 이온성 재료에 대한 CHGNet **검증**

### 실용적 응용

1. 시멘트 생산에서 **90% CO₂ 감소**
2. **빠른 R&D 사이클** (년 → 월)
3. **비용 절감** (실험 시행착오 최소화)
4. **지속 가능한 건설** 재료

### 프로젝트 산출물

- **코드**: 오픈소스 저장소
- **데이터**: MLP 예측 특성의 공유 가능한 데이터셋
- **데모**: 결과의 대화형 시각화
