## 핵심 발견

### 1. 최적 Ca/Si 비율: 2.5

```
Ca/Si = 2.5에서 가장 큰 수화 에너지 변화 관찰
ΔE = -47.64 eV (가장 안정화)
```

- Ca/Si 2.5는 C₂S와 C₃S 사이의 중간 조성
- 적절한 Ca 함량 + Si 반응 사이트 균형

### 2. Ca/Si와 수화 반응성 관계

```
Ca/Si 1.5 → 2.0 → 2.5: 수화 반응성 증가
Ca/Si 2.5 → 3.0: 수화 반응성 감소
```

- 너무 낮은 Ca/Si (1.5): Ca 부족으로 수화 제한
- 너무 높은 Ca/Si (3.0): Si 부족으로 반응 사이트 감소
- 최적점: Ca/Si = 2.5

### 3. 구조 안정성 (E/atom)

| Ca/Si | E/atom (eV) | 안정성 |
|:-----:|:-----------:|--------|
| 1.5 | -6.46 | 낮음 |
| 2.0 | -6.81 | 중간 |
| 2.5 | -7.07 | 높음 |
| 3.0 | -7.11 | 가장 높음 |

- Ca 함량이 높을수록 구조적으로 안정
- Ca/Si = 3.0이 가장 안정적이지만, 수화는 2.5가 최적

### 4. 기존 MatterGen 구조 문제점 추론

기존 MatterGen 구조가 수화 성능이 낮았던 이유:
- Ca/Si 비율이 낮았을 가능성 (< 2.0)
- Ca-poor 조성으로 수화 반응성 제한
