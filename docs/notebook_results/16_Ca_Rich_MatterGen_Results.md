---

## 실험 개요

### 배경
- 15번 노트북에서 **최적 Ca/Si = 2.5** 확인 (ΔE = -47.64 eV)
- 기존 MatterGen 구조의 낮은 수화 성능 원인: Ca/Si 부족 추정
- Ca-rich 구조로 수화 성능 개선 가능성 검증

### 실험 조건

| 구조 | Ca/Si | 화학식 유형 | 원자 수 |

## 핵심 발견

### 1. Ca-rich 구조의 개선 효과

```
기존 MatterGen 수화 점수: 48~54점
Ca-rich 구조 수화 ΔE: -49.6 eV (C3S_like)

→ Ca/Si 비율 증가로 수화 반응성 개선됨
```

### 2. 산업폐기물에 미치지 못하는 이유

| 요인 | 산업폐기물 | Ca-rich 구조 |
|------|-----------|--------------|
| 조성 | Ca-Si-Al-Fe-Mg-Na-O | Ca-Si-O |
| 복잡도 | 높음 (다원소) | 낮음 (3원소) |
| 구조 | 실제 공정 최적화 | 랜덤 생성 |
| 결정성 | 부분 비정질 | 완전 비정질 |

**결론**: 단순 Ca-Si-O 시스템으로는 산업폐기물의 복잡한 수화 특성 재현 불가

### 3. 추가 원소의 중요성

산업폐기물에 포함된 원소별 역할:
- **Al**: 에트링자이트 형성, C-A-S-H 젤 생성
- **Fe**: 수화 촉진, 초기 강도 발현
- **Mg**: 장기 강도 발현
- **Na**: 알칼리 활성화 효과
