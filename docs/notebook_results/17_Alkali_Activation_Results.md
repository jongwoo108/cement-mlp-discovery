# 17. 알칼리 활성화 시뮬레이션 결과

> MatterGen 구조의 대안적 수화 방법 검증

---

## 실험 개요

### 배경
- 노트북 16에서 MatterGen/Ca-rich 구조의 **물 수화 반응성 낮음** 확인
- 하지만 **기계적 강도는 우수** (Portland Cement의 2-3배)
- 알칼리 활성화로 대안적 경로 탐색

### 실험 조건

| 파라미터 | 값 |
|----------|-----|
| 기본 구조 | Ca-Si-O (Ca/Si=3.0, 36 원자) |
| 활성화제 분자 수 | 8개 |
| MD 스텝 | 2000 (2 ps) |
| 온도 | 300 K |

### 테스트 조건

| # | 활성화 방법 | 설명 |
|:-:|------------|------|
| 1 | H₂O | 일반 물 수화 (기준) |
| 2 | NaOH | 수산화나트륨 활성화 |
| 3 | KOH | 수산화칼륨 활성화 |
| 4 | H₂O + NaOH | 혼합 (실제 알칼리 용액) |

## 실험 결과

### 활성화 방법별 에너지 변화

| 방법 | 화학식 | E_init (eV) | E_final (eV) | **ΔE (eV)** | 평가 |
|------|--------|:-----------:|:------------:|:-----------:|:----:|
| H₂O | H₁₂Ca₁₂O₂₆Si₄ | -292.8 | -341.6 | -48.9 | ⚠️ |
| NaOH | H₅Ca₁₂Na₅O₂₅Si₄ | -242.0 | -320.3 | -78.3 | ✅ |
| KOH | H₇Ca₁₂K₇O₂₇Si₄ | -259.9 | -351.6 | -91.8 | ✅ |
| **H₂O + NaOH** | **H₁₂Ca₁₂Na₄O₂₈Si₄** | **-251.8** | **-368.4** | **-116.5** | **✅✅** |

### 순위

| 순위 | 방법 | ΔE (eV) | 물 대비 개선 |
|:----:|------|:-------:|:-----------:|
| **1** | **혼합 (H₂O + NaOH)** | **-116.5** | **+138%** |
| 2 | KOH | -91.8 | +88% |
| 3 | NaOH | -78.3 | +60% |
| 4 | H₂O (기준) | -48.9 | - |

---

## 핵심 발견

### 1. 혼합 활성화가 가장 효과적

```
물 수화:       ΔE = -48.9 eV
혼합 활성화:   ΔE = -116.5 eV

개선: -67.6 eV (138% 향상!)
```

### 2. 알칼리 종류별 효과

```
KOH > NaOH > H₂O

- K⁺ 이온이 Na⁺보다 큰 활성화 효과
- 실제 지오폴리머 연구와 일치
```

### 3. 산업폐기물과 비교

| 재료 | 출처 | ΔE (eV) |
|------|------|:-------:|
| EAFSlag | 산업폐기물 | ~-150 |
| **Ca-rich + 알칼리** | **MatterGen** | **-116.5** |
| Ca-rich + 물 | MatterGen | -48.9 |

→ **알칼리 활성화로 산업폐기물 수준에 근접!**

### 연구적 의의

1. **MatterGen 구조의 새로운 활용 경로 발견**
   - 물 수화: ❌ 부적합
   - 알칼리 활성화: ✅ 적합

2. **두 가지 시멘트 대체재 경로 확립**
   - 경로 A: 산업폐기물 + 물 수화 (즉시 사용 가능)
   - 경로 B: MatterGen + 알칼리 활성화 (고강도 옵션)

3. **최적 활성화 조건 확인**
   - NaOH 수용액 (물 + NaOH 혼합)이 최적
   - 순수 알칼리보다 혼합이 더 효과적
