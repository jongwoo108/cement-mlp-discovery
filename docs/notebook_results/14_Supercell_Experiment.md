# 14. Supercell 실험 결과

> MatterGen 구조 Supercell 확장 수화 시뮬레이션

---

## 실험 목적

기존 MatterGen 구조의 **Ca 부족 문제**를 Supercell 확장으로 해결하고자 함.

### 가설

> Supercell 확장 (2×2×2)으로 원자 수를 늘리면  
> 산업폐기물과 유사한 조건에서 공정한 비교 가능

---

## 실험 설정

| 항목 | 값 |
|------|:--:|
| Supercell | 2×2×2 (8배 확장) |
| 선별 기준 | Ca/Si ≥ 1.0 |
| 물 분자 | 원자 수 비례 (최대 25개) |
| MD 시간 | 0.5 ps |
| 온도 | 300 K |

### 선별된 구조

| 원본 | Supercell | 원자 수 |
|------|-----------|:-------:|
| Ca6O2Si2 | Ca48O16Si16 | 80 |
| Ca6O2Si2 | Ca48O16Si16 | 80 |
| Al2Ca2FeO8Si | Al16Ca16Fe8O64Si8 | 112 |

---

---

## 결론

### 실험 결과: 실패

**Supercell 확장이 수화 성능을 개선하지 못하고 오히려 악화시킴**

### 원인 분석

1. **MD 시간 부족** (0.5 ps) - 평형 미도달 가능성
2. **구조적 문제** - 큰 구조에서 물 분자 침투 어려움
3. **선별된 구조의 한계** - Ca6O2Si2가 실제 수화에 불리

### 시사점

> "단순한 구조 확장보다 **조성 최적화**(Ca/Si 비율, 원소 조합)가  
> MatterGen 구조의 수화 성능 개선에 더 중요함"

---

## 향후 연구 방향

1. **Ca-rich 구조 재생성** - MatterGen에 Ca/Si ≥ 2.0 조건 부여
2. **알칼리 활성화 시뮬레이션** - NaOH, KOH 용액 환경
3. **장시간 MD** - 5-10 ps로 평형 도달 확인
4. **다른 화학 시스템** - Ca-Si-O 이외의 조합 탐색
